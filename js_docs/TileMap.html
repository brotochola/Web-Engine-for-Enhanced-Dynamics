Class: TileMap

TileMap(id, name, mapWidth, mapHeight, tileWidth, tileHeight, tilesets)

TileMap - SAB-backed tilemap data accessible from any thread. Static registry mirrors the Layer.js pattern: - Main thread: `TileMap.initializeFromLoaded()` creates SABs from Tiled JSON - Workers: `TileMap.initializeFromBuffers()` creates Int32Array views over shared SABs - All threads: `TileMap.get('name')` returns the same logical tilemap, backed by shared memory
