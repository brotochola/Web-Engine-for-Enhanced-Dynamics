initializeCustomLayers()

Initialize custom layer rendering infrastructure. NON-SHADER LAYERS: ParticleContainer added directly to stage at the layer's zIndex. Camera transform applied via container.scale / container.position. SHADER LAYERS (two-RT pipeline): 1. ParticleContainer rendered (additive blend) → raw density RenderTexture (RT) 2. Fullscreen Mesh with custom fragment shader reads density RT → output RT 3. Output RT displayed on stage via Sprite at the layer's zIndex This enables screen-space effects (metaballs, heat distortion, etc.) driven by entity positions without per-entity shader overhead. Uniforms are shared via Layer SABs with Atomics dirty flags -- any thread can call Layer.water.setUniform('uThreshold', 0.4) and the change is picked up next frame with zero postMessage overhead.
